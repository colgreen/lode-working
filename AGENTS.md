# Agent Instructions

This repository maintains the reusable Lode Working instructions and guidance.

When changing this repo:

- Keep Lode Working tool-agnostic.
- Treat `lode-working.md` as the canonical reusable instruction file.
- Use `memory/` as the default project-memory folder in examples.
- Use `AGENTS.md` as the canonical agent-facing entry point in examples.
- Do not add launcher scripts or agent-specific startup wrappers.
- Keep references to Lode Coding as inspiration, not as the default folder vocabulary.
